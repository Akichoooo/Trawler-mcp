# Trawler-mcp 核心源码词典 (03_CODEBASE_GUIDE.md)

> [!TIP]
> 授人以鱼不如授人以渔。本文档为您梳理了 `trawler/` 目录下所有核心文件的指责边界。无论你是排查 Bug 还是新增功能，都可以迅速锁定修改点。

## 1. 核心入口与胶水层

- **[`trawler/server.py`](trawler/server.py)**: FastMCP 实例的定义与暴露点，注册了 `@mcp.tool()` 给大模型调用。
- **[`trawler/__main__.py`](trawler/__main__.py)**: 程序的启动入口，负责平台判断（引入 `uvloop`）、日志初始化和启动 MCP 服务器。
- **[`trawler/server.py`](trawler/server.py)**: 处理 `stdio` 与 `sse` 双模传输的逻辑分发层，依赖 Starlette。
- **[`trawler/crawl_url.py`](trawler/crawl_url.py)**: **绝对核心**。请求的中央调度器，包含 `crawl_url()` 和 `_do_crawl()`，掌管降级阶梯（Rung0 -> Rung3）和并发限制。

## 2. 爬虫抓取层 (Fetchers)
位于 `trawler/fetcher/` 目录。

- **[`trawler/fetcher/curlcffi_rung.py`](trawler/fetcher/curlcffi_rung.py)**: Rung 0，使用 `curl_cffi` 进行极速、轻量级的 HTTP/2 抓取，自带连接池。
- **[`trawler/fetcher/patchright_rung.py`](trawler/fetcher/patchright_rung.py)**: Rung 1，使用魔改 Chromium 引擎，破盾主力。包含基于贝塞尔曲线的鼠标拟人移动。
- **[`trawler/fetcher/jina_rung.py`](trawler/fetcher/jina_rung.py)**: Rung 2，调用 `r.jina.ai` 接口的服务商代工层。
- **[`trawler/fetcher/hitl_rung.py`](trawler/fetcher/hitl_rung.py)**: Rung 3，真人协助介入层（Human-in-the-Loop）。
- **[`trawler/fetcher/detect.py`](trawler/fetcher/detect.py)**: 分析返回结果并判断是否遭遇了 Cloudflare, Akamai 等 WAF 的封锁（特征字符串匹配，决定是否降级）。
- **[`trawler/fetcher/challenge_detect.py`](trawler/fetcher/challenge_detect.py)**: 用于侦测页面内是否存在具体的交互式验证码（Turnstile, reCAPTCHA 等）。

## 3. 解析与提取引擎 (Parsers)
位于 `trawler/parser/` 目录，防止脏数据撑爆 LLM 上下文。

- **[`trawler/parser/extract.py`](trawler/parser/extract.py)**: 通过 Readability 算法清洗 HTML，剥离广告和边栏，提取出纯粹的 Markdown 核心内容。
- **[`trawler/parser/extract.py`](trawler/parser/extract.py)**: 正则拦截 Base64 图片等海量数据，防止 BeautifulSoup 解析时发生内存泄漏 (OOM)。
- **[`trawler/parser/title.py`](trawler/parser/title.py)**: 提取网页 `<title>`。

## 4. 数据库与持久化层
高性能、无锁争用的存储基石。

- **[`trawler/db.py`](trawler/db.py)**: SQLite 初始化、PRAGMA 配置（WAL 模式）、建表逻辑。
- **[`trawler/db_writer.py`](trawler/db_writer.py)**: 实现了 `AsyncQueueDBWriter`，承接所有的写操作，在后台单线程批量 COMMIT 避免死锁。
- **[`trawler/atomic.py`](trawler/atomic.py)**: 提供针对文件系统的原子写入（Atomic Write），多进程并发读写原始文件（Raw Files）时防止文件损坏。
- **[`trawler/audit.py`](trawler/audit.py)**: 将所有的爬虫请求、成功/失败原因、拦截器特征记录到 `trawler_audit` 表中。
- **[`trawler/seen.py`](trawler/seen.py)**: URL 去重与缓存校验。
- **[`trawler/raw_store.py`](trawler/raw_store.py)**: 保存网页原始 HTML 文本。

## 5. 安全、策略与周边设施

- **[`trawler/ssrf.py`](trawler/ssrf.py)**: DNS 预检中心，封锁局域网段和云 Metadata，防止利用爬虫打穿内网。
- **[`trawler/urlnorm.py`](trawler/urlnorm.py)**: 健壮的 URL 规范化，去除无效锚点、规范 Hostname、解析根域名。
- **[`trawler/site_rules.py`](trawler/site_rules.py)**: YAML 站点规则匹配引擎，允许针对指定域名下发不同的抓取/渲染策略。
- **[`trawler/robots.py`](trawler/robots.py)**: 遵循 RFC 9309 的 robots.txt 解析与合规缓存，防止恶意触碰蜜罐。
- **[`trawler/errors.py`](trawler/errors.py)**: 全局异常防幻觉契约。定义了所有的 `__TRAWLER_ERROR__` 序列化格式与 `RateLimitError` 异常基类。
- **[`trawler/config.py`](trawler/config.py)**: 从环境变量或默认常量中拉取各种超时与并发阈值。
- **[`trawler/tracing.py`](trawler/tracing.py)**: 分布式链路追踪（Trace ID / Span ID）上下文传递。
- **[`trawler/account_vault.py`](trawler/account_vault.py)**: 用户态凭证和 Cookie `storage_state` 提取与管理。
